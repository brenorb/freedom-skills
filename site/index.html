Freedom technology already exists.

The missing layer is usable knowledge.

Every tool asks people to relearn interfaces, setup, jargon, and risk. Freedom Skills converts that hard-won knowledge into inspectable workflows an agent can carry between compatible environments.

03 / Trust is a boundary

Convenience should not erase consequence.

Good agent workflows make the risky parts legible. They name prerequisites, protect secrets, prefer inspection before action, and stop for human confirmation when an action leaves the machine or cannot be undone.
