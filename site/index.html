The operating model

Instructions travel.
Judgment stays.

  1. 01

    Recognize

    A compatible agent reads lightweight metadata and knows when a skill applies.

  2. 02

    Load

    It opens the detailed SKILL.md only when needed, with scripts and references close at hand.

  3. 03

    Act, visibly

    The workflow names prerequisites, executes the task, and surfaces decisions and caveats to the person in charge.

The trust file

No blank checks.

A skill is an instruction set, not a guarantee. Read the source. Check permissions. Verify the outcome.

01

Inspect

Use Skillspector to scan unfamiliar skills and repositories before installation.

02

Minimize

Grant only the access a workflow needs. Rampart hooks can reduce sensitive-data exposure.

03

Confirm

Messages, payments, publication, and irreversible actions deserve explicit human review.

04

Verify

Check addresses, recipients, hashes, archives, and final state independently.
