SKILL.mdportable by design

Capability that travels

One small package.
Many compatible agents.

An Agent Skill is a portable set of instructions and resources for a specific task. Its metadata helps an agent recognize when it applies; its details load only when useful.

  1. 01
    Discover

    The agent matches a real request to focused metadata.

  2. 02
    Guide

    Instructions, scripts, references, and caveats shape the work.

  3. 03
    Act with you

    You stay in control of consequential choices and external actions.

Trust is cultivated

Useful under real constraints.

High-stakes tools deserve more than confident prose. The library is built to make assumptions visible and improvement observable.

Curated

Specific over sprawling

Each skill is scoped to an actual operation, with prerequisites and limits close at hand.

Tested

Evidence over theater

Scripts and workflows can include examples, checks, benchmarks, and review history.

Safety-aware

Guidance, not false certainty

Risk, trust assumptions, and irreversible actions are surfaced instead of quietly hidden.

Open source

Inspect, adapt, improve

The work is public. People can audit a path, report a gap, and propose a better one.

Proof without surveillance

We measure what the commons produces—not where people click.
