02 / How knowledge travels

Pack the route,
not just the destination.

An Agent Skill is a portable package of instructions and resources. At its heart is a SKILL.md; alongside it can travel scripts, references, templates, and tests.

  1. DiscoverA compatible agent reads compact metadata and recognizes when a skill applies.
  2. LoadDetailed guidance enters context only when it is relevant to the task.
  3. Act with contextThe workflow carries prerequisites, commands, caveats, and trust assumptions with it.

03 / Trust boundary

A useful map marks the cliffs.

Convenience is not permission. Skills should make critical assumptions visible and keep the human in control of consequential actions.

Inspect before installing
Review workflows and scan skill packages. Know what code, services, and permissions are involved.
Secrets stay secret
Credentials, wallet mnemonics, private keys, and sensitive data belong in appropriate local or server-side storage—not source control.
Consent before consequence
Publishing, messaging, paying, and other external actions should remain explicit and intentional.
Caveats travel too
A good skill documents prerequisites, failure modes, privacy tradeoffs, and trust assumptions beside the happy path.
