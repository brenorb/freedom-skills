# Freedom Skills

Freedom Skills is a repository of [AI agent skills](https://agentskills.io) for Bitcoin and other freedom technologies.

It teaches agents how to use tools like Bitcoin for uncensorable payments and Nostr for censorship-resistant communication, so dissidents, activists, and human rights defenders can coordinate and transact without depending on centralized services.

The project focuses on reusable, tested, and benchmarked skills that make freedom tech more legible and usable for AI systems. The long-term goal is to build a trusted layer for skill discovery, validation, and reuse, so agents can operate with more sovereignty in adversarial environments.
