# Event

Route: `https://hostednowhere.com/create/event`

Minimum viable content:

- event name

Success cues:

- the Verification step shows a phrase for the page
- the Share Link step reveals a URL under `https://hostednowhere.com/s#...`

Important caution:

- the builder can publish from a name alone, but real event pages usually need date and venue before sharing

Good demo payload:

- event name: `Lantern Alley Listening Session`
